# Technical Paper Presentation	

## Description
Technical Paper Presentation, an online technical event organised by students of Association of Civil Engineering Students. In this event, participants had to present a technical paper . The event held during the Techical Festival 2021 (**i-GNEz**) of Guru Nanak Dev Engineering College. "i-GNEz" was a four day event in which students from all over India took part.
